# Starter Guid
## Frontend
0. Ensure Having Node:
[Link - GeekforGeeks](https://www.geeksforgeeks.org/installation-guide/install-node-js-on-windows/)
Now go to frontend Directory
```bash
cd frontend
```
1. Ensure Having pnpm:
```bash
npm i pnpm
```
2. Install
```bash
pnpm i
```

3. Start
```bash
pnpm vite dev
```

## Backend
1. go to backend dir
```bash
cd backend
```

2. Init Venv
```bash
python -m venv venv
venv\Scripts\activate
```

3. Install requirements
```bash
python -m pip install -r requirements.txt
```

4. start Django server
```bash
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
